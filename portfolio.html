LIVE & DEPLOYED
Node.js Portfolio on AWS EC2

A simple yet powerful Express.js server that serves this portfolio page. What makes it special isn't the code — it's the infrastructure behind it. Every git push triggers a GitHub Actions pipeline that builds a Docker image, pushes it to a registry, connects to an AWS EC2 instance, pulls the latest container, and restarts the service — fully automated, zero manual steps.

STEP 01
Code Push
Developer pushes code to the GitHub repository, triggering the CI/CD pipeline automatically.
STEP 02
GitHub Actions
Workflow file runs — installs dependencies, builds a Docker image from the Dockerfile.
STEP 03
Docker Container
Image is pushed to Docker Hub registry, containerising the Express.js app with all its dependencies.
STEP 04
Deploy to EC2
Actions SSHes into the AWS EC2 instance, pulls the new image, stops the old container and starts the fresh one.
